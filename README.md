
# Test Play Green

Creation of a test project in React ts for axios consuming the REST APi thesportsdb.com, integration and authentication in the login with Firebase.


## Installation

Install Test Play Green  with npm

```bash
    cd my-project
    npm install
```
RUN

```bash
    npm run dev
```
